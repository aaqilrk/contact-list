# Contact List Web App

Express app with mysql backend.

## Prerequisites

```bash
  nodeJS
  mysql
```

## Installation

Install with npm.

```bash
  git clone https://github.com/Aaqil-Raj-Krishna/contact-list.git 
  cd contact-list
  npm install
  npx sequelize-cli db:create
  npm start
```

Wep app @ [localhost:3000](http://localhost:3000/)
